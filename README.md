# ESP32 Cyberdeck

A portable ESP32-based learning project for hardware hacking, WiFi/BLE scanning and cybersecurity fundamentals.

---

# Goal

Build a small portable cyberdeck-style device to learn:

- ESP32 development
- WiFi scanning
- BLE scanning
- display integration
- hardware wiring
- embedded systems
- cybersecurity fundamentals

---

# Hardware

| Component | Purpose |
|---|---|
| ESP32-C6 | Main microcontroller |
| ST7789 Display | User interface |
| Button | Menu control |
| Battery Module | Portable power |
| Case | Protection and mobility |

---

# Current Features

- WiFi scanner
- BLE scanner
- Display output
- Simple menu navigation
